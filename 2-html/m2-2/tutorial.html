Tutorial de HTML

Uma vez que você conheça a estrutura de um código HTML, as primeiras funções a serem aprendidas são de inserção de texto, imagem e de criar hyperlinks.

Os títulos são colocados com a síntaxe
<h1>(título)</h1>

Além dessa tag "h1", tem tags de títulos de tamanhos menores que vão de "h2" até "h6", o código e o resultado ficam

Código


<h1>título h1</h1>
<h2>título h2</h2>
<h3>título h3</h3>
<h4>título h4</h4>
<h5>título h5</h5>
<h6>título h6</h6>

Resultado

título h1

título h2

título h3

título h4

título h5
título h6
Para inserir um parágrafo, usa-se a tag "p", é possível fazer um páragrafo sem essa tag também. Além disso tem a tag "br" para quebrar linhas, exemplo:

Código


Parágrafo
<p>Parágrafo</p>
<p>Outro parágrafo <br> com quebra de linha </p>

Resultado

parágrafo

parágrafo

Outro parágrafo
com quebra de linha

Para inserir imagem na página, faz:
< img src="(caminho da imagem)">

Código


<img src="cat.jpg">

Resultado

Há o atributo "alt" que faz uma mensagem mostrada caso a imagem não possa ser carregada, assim a sítaxe fica
< ima src="(caminho da imagem)" alt="(mensagem)">
Um exemplo onde isso pode acontecer é caso a imagem não exista, então aparece um ícone com a mensagem do atributo "alt"

Para fazer um hyperlink usamos
<a href="(endereço da página web)">"(palvra/frase linkada)"</a>

Código


<a href="https://www.google.com.br/">Google</a>
<a href="https://www.google.com.br/">Dá pra linkar frases também</a>

Resultado


Google
Dá pra linkar frases também

Para fazer uma lista usa-se a tag "ul" para abrir as listas e os itens das listas são colocados nas tags "li"

Código


<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Resultado


  • Item 1
  • Item 2
  • Item 3
Podemos fazer uma lista enumerada apenas mudando a tag "ul" para "ol"

Código


<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

Resultado


  1. Item 1
  2. Item 2
  3. Item 3
Podemos fazer umas misturas como linkar uma imagem ou fazer listas de títulos. Um outro atributo das imagens é o "title" que faz tooltip, que é um texto que aparece quando se passa o cursor do mouse pela imagem, a tag com esse atributo fica
<img src="(imagem)" title="(mensagem do tooltip)">
ou incluindo o atributo "alt"
<img src="(imagem)" title="(mensagem do tooltip)" alt="(mensagem do alt)">
Assim encerramos esse tutorial inicial de HTML com exemplos de "misturas" citados
