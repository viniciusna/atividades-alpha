Tutorial 3 de HTML

Continuamos o tutorial agora com alguns inputs para uma página, os inputs é um dos principais meios em que um usuário pode interagir com a página. Veremos aqui os inputs:
  • text
  • password
  • date
  • radio
  • checkbox
  • range
  • reset
  • submit
Para fazer um input usamos a tag "input", o tipo de input é determinado pelo atributo "type", alguns dos valores do type estão listados na lista anterior. Além disso precisamos "dar um nome" para o input, por exemplo, indicar numa caixa de texto que o usuário deve colocar o seu nome. Para isso, usamos a tag "label".

Código

<label> Nome </label> <input type="text" />

Resultado

Além disso podemos identificar uma tag com o atributo "id" e então escolher um nome para identificar aquela tag, isso é útil para o input, o elemento ficaria assim:
<input type="text" id=" (identificador da tag) " />
Isso é bem útil ao ser utilizado juntamente com o atributo "for" da tag "label", colocamos o valor do "id" da tag "input" no valor do atributo "for" da tag "label" e assim, quando o usuário clicar na palavra/frase que esteja sob o label fará com que o cursor vá para o input com a "id" que está no "for".

Código

<label for="teste"> clique por aqui </label> <input type="text" id="teste" />

Resultado

Seguiremos esse padrão para fazer nossos inputs aqui. Nesse caso então, o que vai mudar de um exemplo para outro a seguir é apenas o valor do atributo "type" na tag "input"

1- Input "text"

Esse a gente já viu nos exemplos anteriores, esse input abre uma caixa de texto, do tipo que a gente geralmente vê para inserir nome, email, senhas e etc.

2- Input "password"

Esse é um input tipo "text" mas não mostra os caracteres que foram digitados, ao invés disso, ele mostra pontos pretos, é utilizados em campos que se insere senhas. Exemplo

Código

<label for="email"> Email </label> <input type="text" id='email' />
<label for="senha"> Senha </label> <input type="password" id='senha'>

Resultado


3- Input "date"

Esse serve pra colocar um espaço para se colocar data, ele já vem formatado para para colocar data e tem um calendário embutido.

Código

<label for="data"> Data de nascimento </label> <input type="date" id="data" />

Resultados

4- Input "radio"

O "radio" é uma opção de marcar muito visto em formulários, para fazer perguntas que tenham alternativas. Podemos usar o atributo "name" para vincular as alternativas a uma mesma questão, assim somente uma das alternativas podem ser marcadas.

Código

<label for="a"> Alternativa a </label> <input type="radio" id="a" name="pergunta">
<label for="b"> Alternativa b </label> <input type="radio" id="b" name="pergunta">
<label for="c"> Alternativa c </label> <input type="radio" id="c" name="pergunta">

Resultado

5- Input "checkbox"

A checkbox é usado principalmente em questões de formulários que permite marcar mais de uma alternativa, diferente da "radio", a checkbox pode ser desmarcada.

Código

<label for="c1"> Checkbox 1 </label> <input type="checkbox" id="c1" />
<label for="c2"> Checkbox 2 </label> <input type="checkbox" id="c2" />

Resultado

6- Input "range"

Esse serve pra colocar uma barra que o usuário pode controlar para controlar algum valor, parecido com a barra de volume de apps

Código

Volume <input type="range" />

Resultado

Volume
