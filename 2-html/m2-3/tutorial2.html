Conhecido as funções básicas, podemos avançar para algumas estilizações, como mudar a cor do texto, para isso se usa o código de cores RGB que é explicado nesse link. Depois de entendido esse código RGB, usamos esse código no atributo "style" que é um atributo para parágrafos, títulos e outras tags que contenham textos, no caso, o atributo é colocado na tag de abertura
<p style="color:(código RGB);"> (texto) </p>
Cores "definidas" que tenham nome como vermelho, verde, aqua, cinza e etc, não precisa colocar o código RGB, pode colocar simplesmente o nome da cor em inglês ao invés do seu código. Alguns exemplos de seu uso

Código


    <p style="color: #ff0000;"> Parágrafo em vermelho </p> 

    <p style="color: red;"> Outro parágrafo em vermelho </p> 

    <h2 style="color: aqua;"> Título h2 </h2> 

    <h3 style="color: #012345;"> Título h3 </h3> 

    <h4 style="color: rgb(18, 131, 62);"> Título h4 </h4>

Resultados


Parágrafo em vermelho


Outro parágrafo em vermelho


Título h2

Título h3

Título h4

Para destacar palavras em itálico usa-se a tag "i", para negrito usa-se as tags "strong" ou "b"

Código


    <i> Itálico </i> 

    <b> Negrito com a tag "b" </b> 

    <strong> Negrito com a tag "strong" </strong> 

    <b style="color: orangered;"> Negrito colorido </b> <i style="color: rgb(21, 125, 151);"> e itálico colorido </i> 

    <b><i> Negrito e itálico </i></b>

Resultados


Itálico
Negrito com a tag "b"
Negrito com a tag "strong"
Negrito colorido e itálico colorido
Negrito e itálico

Esses atributos de texto também funcionam em listas de forma análoga, no caso de listas temos duas tags, "ul" e "li", o atributo na tag "ul", será aplicado em todos os itens da lista, enquanto que o atributo na tag "li" vai aplicar aquele atributo apenas naquele item da lista

Código


    <ul style="color: darkblue;"> 

        <li> Item 1 </li> 

        <li> Item 2 </li> 

        <li> <i> Item 3 </i> </li> 

    </ul> 

        

    <ul> 

        <li style="color: green;"> Primeiro </li> 

        <li style="color: yellow;"> <i> Segundo </i> </li> 

        <li style="color: red;"> <b> Terceiro </b> </li> 

    </ul>

Resultado


  • Item 1
  • Item 2
  • Item 3
  • Primeiro
  • Segundo
  • Terceiro
O link no primeiro parágrafo desse tutorial abre uma página numa nova aba do navegador, para fazer isso usamos o atributo "target" na tag "a", o valor do atributo é "blank", o elemento daquele link fica
<a href="https://dicasdeprogramacao.com.br/entenda-como-funcionam-os-codigos-de-cores-rgb/" target="blank">link</a>. Mais exemplos:

Código


    <a href="https://www.google.com.br/"> Tag sem "target" abre a página na mesma aba </a> 

    <a href="https://www.google.com.br/" target="blank"> Tag com "target" igual a "blank" abre a página numa nova aba </a>

Resultado

Tag sem "target" abre a página na mesma aba
Tag com "target" igual a "blank" abre a página numa nova aba

Para fazer tabelas utiliza a tag "table", dentro dessa tag há outras duas que são usadas para organizar os itens na tabela, tem a tag "tr" usada para criar uma linha na tabela, dentro dessa tag colocamos a tag "th" para montar as colunas e dentro dessa tag colocamos os itens da tabela

Código


    <table> 

        <tr> 

            <th> Item 1.1 </th> 

            <th> Item 1.2 </th> 

            <th> Item 1.3 </th> 

            <th> Item 1.4 </th> 

        </tr> 

        <tr> 

            <th> Item 2.1 </th> 

            <th> Item 2.2 </th> 

            <th> Item 2.3 </th> 

            <th> Item 2.4 </th> 

        </tr> 

        <tr> 

            <th> Item 3.1 </th> 

            <th> Item 3.2 </th> 

            <th> Item 3.3 </th> 

            <th> Item 3.4 </th> 

        </tr> 

    </table>

Resultado


Item 1.1 Item 1.2 Item 1.3 Item 1.4
Item 2.1 Item 2.2 Item 2.3 Item 2.4
Item 3.1 Item 3.2 Item 3.3 Item 3.4
