Tutorial de HTML


Iframe


Iframe são aquelas janelas que vemos nas páginas que contém conteúdo de outra página ou de um vídeo. Para isso usamos a tag "iframe" da seguinte forma

<iframe src="(página a ser mostrada)"></iframe>

Código

        <iframe src="https://pt.wikipedia.org/wiki/Alan_Turing"></iframe>

Resultado

Podemos utilizar atributos "width" e "height" para manipular o tamanho da janela do iframe, "frameborder" para o tamanho da borda da janela

Código

        <iframe src="https://pt.wikipedia.org/wiki/Alan_Turing" frameborder="0" width="500px" height="400px"></iframe>

Resultado

Para colocar um vídeo do YouTube ou uma aplicação de algum site como uma janela iframe com o google maps dentro, vai em compartilhar e em uma das opções será "incorporar" onde já vem uma tag pré-definida pra colocar no HTML, mas também pode-se pegar o link do vídeo "embed" nessa tag. Para vídeos temos o atributo "allowfullscreen" para permitir abrir o vídeo em tela cheia.

Código

        <iframe src="https://www.youtube.com/embed/kn3ss_0J4W8" frameborder="0" width="560" height="315" allowfullscreen></iframe>

Resultado
